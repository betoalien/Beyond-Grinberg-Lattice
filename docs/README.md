# docs/

GitHub Pages source using **Just the Docs**.

## Recommended GitHub Pages settings
In the GitHub repository settings:
1. Go to **Settings -> Pages**
2. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/docs`

## Notes
- Site config is in `docs/_config.yml`
- The PDF download served by Pages is stored in `docs/assets/downloads/`
- External links (arXiv / alphaXiv) can be added later in `docs/links.md`
